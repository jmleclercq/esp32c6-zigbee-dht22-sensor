# ESP32-C6 Zigbee DHT22 Sensor

## 📖 Project Overview

This project aims to build a fully functional Zigbee temperature and humidity sensor using:

- ESP32-C6 (Waveshare ESP32-C6-Zero)
- DHT22 temperature & humidity sensor
- ESP-IDF + ESP Zigbee SDK
- Home Assistant (ZHA integration)

The goal of this project is not only to build a working device, but also to understand how Zigbee works at a low level using the ESP32-C6 platform.

---

## 🎯 Objectives

- [ ] Create a working Zigbee End Device
- [ ] Implement Temperature Measurement cluster (0x0402)
- [ ] Implement Relative Humidity Measurement cluster (0x0405)
- [ ] Successfully join a ZHA network
- [ ] Validate stability and reporting behavior
- [ ] Optimize for low power usage
- [ ] Design a dedicated enclosure
- [ ] Explore alternative power supply solutions
- [ ] Fully document the development process

---

## 🧰 Hardware

- Waveshare ESP32-C6-Zero
- DHT22 temperature and humidity sensor
- USB-C cable
- Zigbee coordinator compatible with ZHA
- Breadboard and jumper wires

---

## 🔧 Software Stack

- ESP-IDF
- ESP Zigbee SDK
- Home Assistant
- ZHA (Zigbee Home Automation integration)
